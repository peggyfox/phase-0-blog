> Display: Inline v. Block
June 15, 2014

Inline display indicates that an element will be displayed in the same line as another element. Block display indicates that the element will be displayed on the next line down.


Uses

Inline is considered useful for small pieces of content, such as a few words. It can also be useful for images, icons, or buttons (such as navigation) that you'd like to display on the same line. In the case of inline, dimensions are meaningless. Spans are by default set to display inline.

Block is used to ensure that content appears on the next line. Dimensions (width and height) can be specified. Divs are by default set to disply block.
